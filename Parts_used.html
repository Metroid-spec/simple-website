Arduino Smart Car Components

The following components were used to create the device

  • ELEGOO UNO R3 (any arduino board will do)
  • DC L298N Motor Drive Controller Module with DC Motor and Car Wheel
  • G90 9g Micro Servos Motor
  • Ultrasonic Sensor Module Kit HC-SR04 Distance Sensor
  • 2 in. Soft Rubber Swivel Plate Caster
  • 2 in. Breadboard Set Prototype Board
  • AA Battery Holder
  • A solid body to attach everything to
  • Jumper cables
  • Batteries
